Documentation

To generate the documentation HTML pages automatically, you'll have to install doxygen tool first to make the make html command work.

All the variables, functions or attributes that starts with an underscore (_) is meant to be private for that specific file, in case of functions/variables, or for the struct, in case of struct attributes. Even if it's possible access them, they aren't supposed to be changed nor read directly.
